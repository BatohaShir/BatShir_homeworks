Относительная единица em. Относительна размеру шрифта

текст по стандарту равен = 16px
font-size задан: 1em => 1 * 16 = 16px
font-size задан: 1.5em => 1.5 * 16 = 24px
свойство font-size потомка наследует свойство font-size своего родителя, т.е это будет равно: 1.5em => 1.5 * 16 = 24px
здесь немного интереснее em расчитывается по формуле: (свойство потомка * размер шрифта родителя = размер шрифта в пикселях), 1.5em * 24px = 36px
задается: 0.8em * 24px = 25.5px он так же форматируется по формуле (потомок * результат родительского = размер шрифта в пикселях)
width : 6.25em
height : 6.25em
padding: 0.9375em;

Относительная единица rem. Относительна размеру шрифта браузера

текст по стандарту равен = 16px
font-size задан: 1rem => 1 * 16 = 16px
font-size задан: 1.5rem => 1.5 * 16 = 24px
свойство font-size элемента наследует свойство font-size всего html документа, т.е это будет равно: 1.5rem => 1.5 * 16 = 24px
здесь уже отличается от единицы "em", rem не зависит от размера шрифта родительского элемента: (элемент * размер шрифта html документа = размер шрифта в пикселях), 1.5em * 16px = 24px
width : 6.25rem
height : 6.25rem
padding: 0.9375rem;
width : 6.25rem
height : 6.25rem

% проценты от ... Относительная едицина. Разные CSS свойства вычисляют процент относительно разных значений.

Задаётся font-size: 100% => 16 / 100 * 100 = 16px
Задаётся font-size: 150% => 16 / 150 * 100 = 24px
Наследуется font-size от родительского элемента: Задаётся font-size: 150% => 16 / 150 * 100 = 24px
Здесь немного интереснее: (размер шрифта родителя / 100% * свойство потомка = размер в пикселях), 24 / 100 * 150 = 36px
